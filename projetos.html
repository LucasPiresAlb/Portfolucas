Sistema de Automação com Macros - Mobit

Reconhecimento:

Contexto do Projeto

Durante minha atuação na Mobit, identifiquei um processo crítico e repetitivo que consumia horas de trabalho manual da equipe, gerando frustração e limitando o foco em atividades estratégicas.

Solução Desenvolvida

Desenvolvi um conjunto de macros em VBA (Visual Basic for Applications) que automatizou completamente o processo de classificação e análise de dados. A solução incluiu:

  • Análise automática de dados visuais de equipamentos de fiscalização
  • Classificação inteligente de registros com base em parâmetros predefinidos

PROJETO ACADÊMICO

Desenrola Aí - API de Integração de Sistemas

Repositório:

github.com/LucasPiresAlb/Desenrola-AI

Sobre o Projeto

Plataforma desenvolvida como projeto acadêmico da disciplina de Integração de Sistemas que conecta prestadores de serviços e clientes em Fortaleza. O sistema demonstra comunicação robusta entre múltiplas APIs, incluindo geocodificação e notificações em tempo real.

Arquitetura e Funcionalidades

O projeto implementa uma arquitetura de microsserviços com integração entre sistemas externos:

  • API Principal (Node.js/Express): Gerencia solicitações de serviços, prestadores e clientes
  • API de Notificações: Sistema independente para envio de notificações assíncronas
  • Integração com Geocodificação: Conversão automática de endereços em coordenadas geográficas
  • Testes Automatizados: Suite completa com Jest e Supertest garantindo qualidade
  • Documentação Completa: Coleção Postman e documentação de endpoints

Fluxo de Integração

O sistema processa solicitações de serviço através de um fluxo completo:

  1. Validação: Verifica se o prestador existe e se os dados são válidos
  2. Geocodificação: Converte o endereço em coordenadas via API externa (geocode.maps.co)
  3. Persistência: Armazena a solicitação com dados enriquecidos no banco de dados
  4. Notificação: Comunica com API de notificações para alertar o prestador de serviço
  5. Confirmação: Retorna sucesso ao cliente com detalhes da solicitação criada

Stack Tecnológica

Node.js Express.js Jest Supertest Postman Geocoding API RESTful API Microsserviços

Aprendizados

Este projeto consolidou conhecimentos em:

  • Integração entre múltiplos sistemas e APIs externas
  • Arquitetura de microsserviços e comunicação assíncrona
  • Desenvolvimento orientado a testes (TDD)
  • Documentação técnica e boas práticas de desenvolvimento
  • Tratamento de erros e validações robustas
